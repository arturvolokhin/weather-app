# Weather-app
### Данное приложение использует [REST API](https://weatherstack.com/).<br>

***

### *Функционал приложения:*

* При запуске приложения - запрашивается разрешение пользователя на обнаружение местоположения,
          затем возвращается прогноз по местоположению пользователя.  
* В строке поиска вводится нужный город латинскими буквами, либо нужные координаты, затем 
          информация по найденному городу выводится на главном экране.
* Информация о каждом городе сохраняется в модальном окне "History".<br>
  При запросе уже существующего города в модальном окне - происходит обновление данных о данном городе.  
* Кнопка "My location" находит ваше местоположение и возвращает прогноз.
* Информация о последних запросах может быть очищена нажатием кнопки "Clear".



### *Используемые технологии:*

* Приложение написано на нативном **Js**
* Используется **БЭМ** и препроцессор **Sass**.
* Для сборки проекта настроен и использован сборщик **Webpack**.
* Настроен и использован **esLint**.
* Присутствует адаптация под все устройства.
